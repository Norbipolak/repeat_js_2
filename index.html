Document Object Model

A Document Object Model (DOM) a HTML dokumentum reprezentációja. A DOM-on keresztül tudjuk elérni a HTML dokumentum egyes részeit.

Elemhez való hozzáférés

  • document.getElementById: Lement egy elemet az ID alapján
  • document.querySelector: Hogyha a # alkalmazzuk az ID neve előtt, akkor ID alapján menti le az első elemet, amit megtalál
